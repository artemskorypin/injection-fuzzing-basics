# Server-Side Template Injection (SSTI) wordlists

Словари для обнаружения **инъекций на уровне шаблонизаторов** (Server-Side Template Injection).

Фокус тот же, что и во всём репозитории:

> **одна строка = один payload**, упор на детект и первичную разведку.  
> За сложной эксплуатацией (чтение файлов, RCE, обход sandbox и т.п.) — см. раздел *Server Side Template Injection* в PayloadsAllTheThings.

---

## Файлы

### `SSTI_polyglot.txt`

Мини-словарь (фактически одна полиглотная строка) для **общего детекта SSTI**, когда:

- неизвестно, какой движок используется;
- неизвестно, в каком именно контексте рендерится ввод.

Идея полиглота:

- содержит фрагменты синтаксиса разных шаблонизаторов (`${}`, `{{}}`, `<% %>`, кавычки, `%` и т.п.);
- если шаблон действительно исполняется, часто видно:
  - ошибку рендеринга,
  - или “подъедание” части строки шаблонизатором.

Использовать как **первый выстрел**: просто подставить значение из файла и посмотреть на реакцию ответа/логов.

---

### `SSTI_engines_detection.txt`

Небольшой набор выражений для **фингерпринта движка**:

- простые математические выражения вроде `{{7*7}}`, `${3*3}`, `<%= 7*7 %>` и т.д.;
- разные шаблонизаторы по-разному:
  - либо возвращают результат (`49`, `9`),
  - либо выбрасывают специфичную ошибку,
  - либо печатают строку как есть.

С помощью этих payload’ов можно:

- после общего детекта (`SSTI_polyglot.txt`) прикинуть, что за движок крутится (Jinja2, Twig, Freemarker, Thymeleaf, Smarty, ERB, Razor, Velocity и т.п.);
- дальше уже идти в PayloadsAllTheThings и подбирать эксплуатационные примеры именно под этот движок.

---

## Использование

- Оба файла удобно подключать как payload-листы в ZAP/Burp/любой другой фаззер.
- Типичный порядок:
  1. Прогнать `SSTI_polyglot.txt` по подозрительным параметрам/полям.
  2. Если есть признаки интерпретации шаблона — прогнать `SSTI_engines_detection.txt`, чтобы сузить круг движков.
  3. После этого уже обращаться к внешним источникам (PayloadsAllTheThings) за более тяжёлыми payload’ами под конкретный шаблонизатор.
