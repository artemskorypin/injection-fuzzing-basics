# XSS Polyglot payloads

> **одна строка = один payload**

Папка `Polyglot` содержит словари **полиглотных XSS-нагрузок** — строк, которые пытаются одновременно работать в нескольких контекстах:

- HTML / атрибуты
- JS-строки / JS-код
- URL / данные внутри параметров
- иногда ещё и в SVG/JSON и т.п.

Идея: такие payload’ы особенно полезны при **фаззинге неизвестного контекста**, когда заранее непонятно, куда именно попадает ввод.

---

## Файлы

### `XSS-POLYGLOTS.txt`

- Небольшой, более «аккуратный» набор полиглотов.
- Каждая строка — отдельная идея: обход фильтров, игра с кавычками, комментариями, евент-хендлерами и т.д.
- Хорошо подходит:
  - для быстрых проверок,
  - для использования как custom payload-лист в ZAP/Burp, когда не хочется заливать сотни строк.

Рекомендуется как **основной словарь полиглотов**.
